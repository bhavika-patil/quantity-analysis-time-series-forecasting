# Quantity Analysis Using Time Series Data

## Project Overview

This project analyzes quantity sales data using Time Series Analysis and ARIMA forecasting techniques.

## Technologies Used

* Python
* Pandas
* NumPy
* Matplotlib
* Statsmodels

## Workflow

1. Exploratory Data Analysis
2. Time Series Decomposition
3. ARIMA Modeling
4. Evaluation and Visualization

## Results

* Built an ARIMA forecasting model
* Generated future sales predictions
* Evaluated model performance using MSE
